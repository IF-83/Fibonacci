This Fibonacci calculator is almost as cool as the previous two combined.

Fibonacci numbers are defined by the recursion Fn = Fn-1 + Fn-2 starting with F0 = 0 and F1 = 1. The first few numbers are: 0, 1, 1, 2, 3, 5, 8, 13, 21, 35, 56...
Though there is a simple mathematical formula for the nth Fibonacci number, it involves the calculation of the nth power of irrational numbers, which has very high inaccuracy. With this calculator you can find the exact integer value of Fibonacci numbers up to a few millions under an acceptable time frame. You can also choose your preferred algorithm to use and check how fast it is.



Simple recursive function that calls itself for n-1 and n-2. It calculates each value many times, therefore it has exponential complexity. Not a good idea for n greater than 20.

A static array is initialized as [0,1]. When the function is called it returns the nth number of the array if it exists. If not, a new value is appended and the function calls itself with the same parameter. Time complexity is O(n2), space complexity is O(n3) with the n frames that can be eliminated by tail call optimization.

An array is filled with one loop and then the last element is returned. This is an improved version of the previous one without recursion therefor is has O(n2) time and space complexity.

The most straightforward algorithm. A loop of length n calculates the nth number without creating an array for the full sequence (only the last 2 numbers are stored). Time complexity is O(n2) time and O(n) space.

Mathematical observation: the powers of the 2×2 matrix M((0, 1), (1, 1)) contain the Fibonacci numbers. The nth power of the matrix can be calculated with repeated squaring with a logarithmic number of multiplications. Complexity is O(nlog(n)) and O(n) space.
